  📱 COMMENT ÇA MARCHE ?
═══════════════════════════════════════════════════════════════════════════

SUR MOBILE (largeur ≤ 768px) :
1. Un bouton burger (☰) apparaît en haut à droite
2. Cliquez dessus → le menu s'ouvre
3. Le burger devient un X
4. Cliquez sur un lien → le menu se ferme
5. Cliquez en dehors → le menu se ferme

SUR DESKTOP (largeur > 768px) :
1. Le menu est visible en horizontal
2. Tous les liens sont affichés
3. Pas de bouton burger
4. Navigation classique

  🔧 DÉPANNAGE

Le menu ne s'ouvre pas ?

→ Vérifiez que assets/css/style.css est chargé
→ Videz le cache (Ctrl + F5)

Le burger ne devient pas X ?
→ Vérifiez que le CSS est bien chargé
→ Videz le cache du navigateur
