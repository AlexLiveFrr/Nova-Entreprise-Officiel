💡 UTILISATION RAPIDE :

Pour ajouter des animations sur une nouvelle page :

1. Inclure dans le <head> :
   <link rel="stylesheet" href="assets/css/animations.css">

2. Inclure avant </body> :
   <script src="assets/js/rewards.js"></script>

3. Utiliser les classes :
   <div class="animate-fade-in-up hover-lift">
       Votre contenu
   </div>
